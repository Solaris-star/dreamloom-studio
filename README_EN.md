# Dreamloom Studio

[English](README_EN.md) | [中文](README.md)

Dreamloom Studio is a local web writing tool for novel writers, built with Vue 3 and Vite. It includes bookshelf management, chapter editing, character profiles, map design, relationship graphs, dictionaries, AI-assisted writing, and local novel download utilities.

## Features

- Book management: create, edit, delete, and organize local books.
- Writing editor: TipTap-based rich text editing with autosave, word count, themes, and shortcuts.
- Story assets: character profiles, dictionaries, timelines, event charts, organization charts, and relationship graphs.
- Map design: Canvas-based tools for drawing, shapes, text, assets, zooming, undo, and export.
- AI assistance: naming, polishing, continuation, cover images, character images, and scene images with user-provided provider keys.
- Novel download: search by title or author, then save locally or export as TXT. Please follow copyright rules and local laws.

## Stack

- Vue 3
- Vite
- Element Plus
- Pinia
- TipTap
- ECharts
