# Dreamloom Studio

[English](README_EN.md) | [中文](README.md)

Dreamloom Studio 是一款面向小说作者的本地 Web 写作工具，基于 Vue 3 与 Vite 开发。它提供书架管理、章节编辑、人物档案、地图设计、关系图谱、词条字典、AI 辅助写作和小说下载等功能，适合在本机浏览器中使用。

## 功能

- 多书籍管理：支持书籍创建、编辑、删除、封面设置和本地数据保存。
- 写作编辑器：基于 TipTap，支持富文本编辑、自动保存、字数统计、主题设置和快捷键。
- 创作资料管理：支持人物档案、词条字典、时间线、事序图、组织架构和关系图谱。
- 地图设计：基于 Canvas，支持画笔、形状、文字、资源图标、缩放、撤销和导出。
- AI 辅助：支持起名、润色、续写、封面图、人物图和场景图，相关能力需要自行配置服务商 Key。
- 小说下载：支持按书名或作者搜索，将内容下载到本地或导出为 TXT。请遵守版权和当地法规。

## 技术栈

- Vue 3
- Vite
- Element Plus
- Pinia
- TipTap
- ECharts

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run start
```
